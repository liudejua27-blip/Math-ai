# math 文献库说明

本目录用于保存与“高中数学思维 Agent / Math-SEARAG / 数学私教 WebApp”相关的论文资料。当前已经按论文用途整理为 4 个主题文件夹，便于后续写论文、做综述、设计检索增强模块时快速取材。

## 01_数学表达式识别

用途：支撑论文中“数学文本、公式、手写过程识别”的技术背景，也服务于 App 中题目解析、学生草稿识别、解题步骤抽取等模块。

- `复杂数学表达式识别：基准测试、大规模数据集和强基线.pdf`：数学表达式识别 benchmark，可用于说明数据集、评价指标和强基线。
- `基于掩码离散扩散的符号感知推理用于手写数学表达式识别.pdf`：面向手写数学表达式识别，适合作为“学生草稿/手写输入”方向参考。
- `一种用于数学表达式识别的混合视觉Transformer方法.pdf`：视觉 Transformer 识别数学表达式，可作为模型架构参考。
- `来自 zbMATH Open 的 LLM 支持的文档分离功能，用于打印评论.pdf`：数学文档分割与结构化处理参考，适合放在数学知识库构建部分。

## 02_RAG与文档理解

用途：支撑 Math-SEARAG 的核心方法，包括文档转换、结构化证据检索、多模态 RAG、GraphRAG、长文档理解和文档问答。

- `ARIA：自适应检索智能助手——面向领域特定工程教育的多模态 RAG 框架.pdf`：领域教育场景的多模态 RAG 框架，和本项目教育 Agent 主线贴近。
- `CodaRAG：运用联想思维连接点滴，灵感源于互补学习.pdf`：联想式检索增强，可用于“启发式追问/知识关联”设计。
- `DocSeeker：基于证据的结构化视觉推理，助力长篇文档理解.pdf`：长文档证据抽取和视觉推理参考。
- `MAB-DQA：利用多臂老虎机算法解决文档问答中查询方面的重要性问题.pdf`：文档问答中的查询选择策略，可支持自适应检索设计。
- `MultiDocFusion：用于增强长篇工业文档 RAG 分类的分层多模态分块流程.pdf`：长文档分块、融合与分类参考，可迁移到教材/讲义/试卷资料处理。
- `从 PDF 到 RAG-Ready：评估面向特定领域问答系统的文档转换框架.pdf`：PDF 到 RAG 知识库的转换评估，是项目资料工程化的重要参考。
- `我们还需要 GraphRAG 吗？针对智能搜索系统的 RAG 和 GraphRAG 基准测试.pdf`：RAG 与 GraphRAG 对比，可用于方法选择和消融实验讨论。

## 03_评测基准与推理

用途：支撑论文中“Agent 能力评估、深度研究评估、复杂推理评估、优化求解评估”的章节，也可转化为 App 后台评价指标。

- `AutoOpt：用于自动化优化问题求解的数据集和统一框架.pdf`：自动化优化问题求解，可作为数学建模/优化题求解参考。
- `DR³ Eval：迈向现实且可复现的深度研究评估.pdf`：深度研究 Agent 评估参考，可用于设计论文实验。
- `MCERF：通过增强检索功能推进工程文档的多模态LLM评估.pdf`：多模态检索增强评估参考。
- `MERRIN：噪声网络环境下多模态证据检索与推理的基准测试.pdf`：复杂证据检索与推理 benchmark，适合支撑“抗噪检索”论述。
- `TangramPuzzle：利用组合空间推理评估多模态大型语言模型.pdf`：组合空间推理评估，可作为数学思维能力评估的类比材料。

## 04_边界参考

用途：这些资料不是高中数学 Agent 的主线，但可能给多模态 Agent、表格理解、偏好反馈或生成式交互提供边界参考。后续如果论文收窄，可以再隔离。

- `超越行到推理：面向多模态电子表格理解和编辑的智能体检索.pdf`：表格理解和编辑 Agent，可用于成绩分析、错题表、学习画像等功能设想。
- `通过相关性反馈协调多维偏好：一种无需训练的文本到图像扩散框架.pdf`：偏好反馈机制参考，和数学教育主线较远，暂作为边界材料保留。

## 05_智能体与检索覆盖前沿

用途：保存与新版“教育 Agent + 深度搜索 + RAG 覆盖率”相关的高价值前沿材料。它们比普通文献更接近新版产品的 Agent 架构和检索质量评估。

- `MTA-Agent：多模态深度搜索代理的开放式方案.pdf`：多模态搜索 Agent 参考，可用于“深度搜题/深度讲解”工作流。
- `Saarthi for AGI：面向形式化验证的领域特定通用智能.pdf`：形式化验证和领域 Agent 参考，可为数学验证路线提供启发。
- `超越相关性：论检索与RAG信息覆盖率之间的关系.pdf`：RAG 不能只看相关性，还要看信息覆盖率；适合写进 Math-SEARAG 评价指标。
