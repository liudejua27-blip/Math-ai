# 企业级 PRD 与产品矩阵

更新时间：2026-05-15

## 企业级定位

产品名称暂定：高考数学私人思维教练。

企业级定位不是“AI 搜题工具”，而是一个面向高中数学学习场景的智能诊断与训练平台：

> 以错题为入口，以证据链为底座，以错因原子为诊断单位，以符号验证保证数学可靠性，以思维档案形成长期个性化学习闭环。

| 管理员 | 确保系统稳定合规 | 权限、审计、安全、内容审核 |

## 产品形态

### 学生端 App / Web

核心闭环：上传错题、确认识别、输入步骤、错因诊断、证据链讲解、变式训练、思维档案。

### 家长端

核心闭环：查看报告、理解孩子短板、获得陪伴建议、购买专项训练包。

### 老师端 SaaS

核心闭环：导入班级试卷、查看共性错因、生成分层作业、追踪学生变化。

### 教研后台

核心闭环：维护知识原子、审核变式题、管理标准解法、复核 AI 输出。

### 企业管理后台

核心闭环：用户、组织、套餐、权限、日志、安全、内容审核、模型配置。

## 核心用户故事

### 学生

作为高三学生，我希望上传一道错题后，系统告诉我从哪一步开始错，而不是只给答案。

验收标准：

- 系统展示学生步骤 S 与正确路径 P 的对齐结果。
- 标出首个错误步骤 E。
- 给出至少一个错因原子 A。
- 每个关键判断引用证据 ID。

### 家长

作为家长，我希望每周看到孩子最常犯的数学思维错误，而不是只看到做题数量。

验收标准：

- 周报包含高频错因、进步点、风险点和建议话术。
- 不展示过度技术细节。
- 对未成年人数据做隐私保护。

### 老师

作为老师，我希望导入一次试卷后看到班级错因分布，并自动生成分层作业。

验收标准：

- 班级报告按题型、章节、错因原子聚合。
- 支持导出 CSV/PDF。
- 推荐作业可人工编辑后发布。

## 产品指标体系

### 学习效果指标

| 指标 | 定义 | 目标 |
| --- | --- | --- |
| Error Localization Accuracy | 错误步骤定位准确率 | Pilot 70%+，正式 85%+ |
| Atom Diagnosis Agreement | 与教师错因标注一致率 | Pilot 70%+，正式 85%+ |
| Variant Pass Rate Lift | 变式题通过率提升 | Pilot 展示趋势，正式显著提升 |
| Mastery Improvement | 原子掌握度提升 | 周维度可追踪 |

### 产品指标

| 指标 | 定义 | 目标 |
| --- | --- | --- |
| Activation Rate | 首次上传并完成诊断比例 | 60%+ |
| Repeat Upload Rate | 7 日内再次上传比例 | 30%+ |
| Report Open Rate | 周报打开率 | 50%+ |
| Paid Conversion | 付费转化率 | 内测后定义 |
| Teacher Retention | 老师端月留存 | 企业试点后定义 |

### 可靠性指标

| 指标 | 定义 | 目标 |
| --- | --- | --- |
| Verification Pass Rate | 关键步骤验证通过比例 | 90%+ |
| Low-confidence Escalation Rate | 低置信度触发人工确认比例 | 可解释可追踪 |
| Unsupported Step Rate | 无证据支撑步骤比例 | 持续下降 |
| Harmful Output Rate | 不当输出比例 | 接近 0 |

## 企业级验收标准

- 产品闭环可由真实学生独立完成。
- 数学输出可追溯到证据链和验证记录。
- 数据和模型调用有审计日志。
- 未成年人隐私保护和家长/学校授权机制明确。
- 老师端支持批量场景，而不是单题玩具 demo。
- 运营后台可配置套餐、模型、审核规则和公告。
