# Math-SEARAG 正式上线生产化清单

当前项目已经达到本地演示和小范围试用 MVP，但要面向中国大陆公开自助访问，还需要按生产系统标准补齐部署、合规、数据安全和发布门禁。

## 上线目标

- 云平台：阿里云。
- 模型：DeepSeek 官方 API。
- 产品形态：公开免费使用，不做付费和邀请码。
- 核心闭环：上传草稿纸/输入步骤 → OCR 确认 → 首错定位 → VerifierTrace → 推荐解法/最快解法 → 图上讲解或 Geometry Lab → 订正/变式 → 学习画像更新。

## 必须完成

- ICP 备案、隐私政策、用户协议、未成年人监护人提示。
- Postgres、Redis、OSS、Python OCR/Verifier、DeepSeek API 的生产环境。
- `/api/readiness` 通过全部生产依赖检查。
- 用户数据导出和删除覆盖聊天、诊断、OCR、学习画像、周报和几何训练。
- OCR 图片和 raw crop 使用私有 OSS，日志脱敏。
- `test:release` 使用严格生产门禁。

## 当前工程已补强

- DeepSeek 官方模型 ID 成为生产默认值。
- 新增 `/api/readiness`。
- 新增 `/api/privacy/export` 与 `/api/privacy/delete`。
- Python 微服务新增 `/health`、`/ocr/draft`、`/verify/math` 生产路径。
- 生产环境默认关闭公开文件上传口，避免草稿纸图片进入公开存储。
- 新增 `/privacy` 与 `/terms` 产品内说明草案。
- 新增生产上线手册：`agent-engineering/frontend/docs/production-launch-cn-20260614.md`。

## 仍需人工确认

- 正式法律文本和公司主体信息。
- 阿里云账号、域名、备案主体、OSS bucket、RDS、Redis、日志服务。
- DeepSeek API key 和模型调用额度。
- 生产 OCR 依赖是否完整安装：Pix2Text、PaddleOCR、LaTeX-OCR。
- OSS 对象删除 worker：当前 API 会删除数据库并返回待删除对象引用，生产环境还需要接入真实 OSS 删除执行器。
