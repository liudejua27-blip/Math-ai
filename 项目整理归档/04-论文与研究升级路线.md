# 论文与研究升级路线

## 1. 新论文定位

旧版论文主线是 Math-SEARAG：公式感知结构化证据增强 RAG。新版应升级为：

> 面向高中数学错题诊断的可验证、可视化、个性化引导式学习 Agent。

推荐中文题目：

> 面向高中数学错题诊断的可视化可验证个性化思维 Agent：基于结构化数学证据、视觉证据链与错因原子记忆的研究

推荐英文题目：

> A Visual, Verifiable, and Personalized Thinking Agent for High-School Mathematics Error Diagnosis with Structured Evidence and Misconception-Atom Memory

## 2. 核心创新点

| 编号 | 创新点 |
| --- | --- |
| C1 | MPES：结构化数学证据表示 |
| C2 | Diagnosis-conditioned SocraticPolicyEngine |
| C3 | VerifierTrace：可验证数学诊断链 |
| C4 | Misconception-atom-centered LearnerMemory |
| C5 | MathCognitiveGraph：可追溯数学认知图谱 |
| C6 | VisualEvidencePipeline：视觉证据确认机制 |
| C7 | 同因变式迁移率评估 |

## 3. 新研究问题

| 编号 | 问题 |
| --- | --- |
| RQ1 | MPES 和 FormulaEvidence 能否提升高中数学题目理解、步骤定位和证据 grounding？ |
| RQ2 | VerifierTrace 能否降低公式幻觉、条件误用和推理跳步？ |
| RQ3 | 诊断条件下的 SocraticPolicy 是否能提升学生自我修正率？ |
| RQ4 | LearnerMemory 是否能降低错因复发率并提升同因变式迁移率？ |
| RQ5 | VisualEvidencePipeline 与 GeometrySceneSpec 是否能提升几何题诊断可信度？ |
| RQ6 | MathCognitiveGraph 是否能提升教师/家长对诊断结果的可解释性和信任度？ |

## 4. 数据集设计

| 数据集 | 规模 | 目的 |
| --- | ---: | --- |
| Dataset A | 100 道导数/函数题 + 学生错误步骤 | 验证首错定位、错因原子、VerifierTrace |
| Dataset B | 100 道立体几何题 + 图形证据 | 验证 VisualEvidence 与 Geometry Lab |
| Dataset C | 300-500 道高中数学主测试集 | 正式主实验 |
| Dataset D | 30-100 名学生使用日志 | 验证自我修正、迁移率和记忆画像 |

## 5. Baseline

| 方法 | 说明 |
| --- | --- |
| Direct LLM | 不检索、不验证，直接讲题 |
| Study-style prompting only | 只用学习模式提示词，不接结构化诊断 |
| OCR + Text RAG | 普通文本切块检索 |
| Formula RAG | 文本 + LaTeX/公式检索 |
| Structured RAG without verifier | 使用 MPES，但无 verifier |
| Math-SEARAG without LearnerMemory | 有诊断，无长期画像 |
| Math-SEARAG without SocraticPolicy | 有诊断，但不做策略化引导 |
| Full Math-SEARAG Learning Agent | MPES + VerifierTrace + SocraticPolicy + LearnerMemory + RemediationLoop |

## 6. 指标

| 维度 | 指标 |
| --- | --- |
| 解题与诊断 | Answer Accuracy, First Wrong Step Accuracy, Misconception Atom Accuracy |
| 证据与可信 | Evidence Faithfulness, Verifier Pass Rate, Unsupported Step Rate |
| 学习过程 | Self-Repair Rate, Student Revision Success |
| 迁移与记忆 | Variant Transfer Rate, Recurrence Rate Reduction |
| 多模态与几何 | Formula Structure Accuracy, Visual Evidence Confirmation Accuracy, Geometry Relation Accuracy |
| 人工一致性 | Teacher Agreement, Cohen's Kappa |
| 运营 | Latency, Cost Per Diagnosis, Human Review Rate |

## 8. 论文红线

不能写：

- 已经证明显著提分。
- 已达到国内最强。
- 模拟实验等同真实教学效果。
- 只展示成功案例。
- 让同一模型生成答案又当裁判。

可以写：

- 原型系统。
- pilot study。
- 可复现评估协议。
- 教师一致性。
- 失败案例分析。
- 对不同题型的收益差异。
