# 研究数据与论文改写计划

日期：2026-06-06

## 1. 当前判断

当前工程 demo 已经具备可行性，但论文还不能直接投稿。战略迁移后，论文必须证明的不只是“诊断准不准”，还要证明：

- 是否遵守无步骤不诊断。
- 是否能定位第一断点。
- 是否能通过 SocraticPolicy 提升自我修正率。
- 是否能通过 LearnerMemory 降低错因复发率。
- 是否能通过 VerifierTrace 提升可信度。
- 是否能通过同因变式验证迁移。

旧的黑盒模拟结果只能作为工程调试记录，不能作为正式论文结果。

## 2. 目标论文结构

1. Abstract：问题、方法、实验、结果，结果没有真实数字时用 `[待填]`。
2. Introduction：高中数学错题诊断痛点、现有搜题/RAG/LLM 家教不足、本文贡献。
3. Related Work：AI tutoring、Study Mode、DeepTutor、数学 OCR/解析、RAG/GraphRAG、多模态数学、形式化验证。
4. Method：MPES、SocraticPolicy、LearnerMemory、VerifierTrace、VisualEvidence、FormulaEvidence、MathCognitiveGraph、RemediationLoop。
5. Experiments：数据集、标注协议、baseline、指标、统计方法。
6. Results：主结果、消融、题型分层、失败案例、人工一致性。
7. Discussion：适用边界、失败模式、教学意义、部署与合规。
8. Conclusion：贡献、边界、未来工作。
9. Appendix：schema、标注规范、提示词原则、更多案例、日志字段。

## 3. 数据集规模

| 数据集 | 规模 | 用途 |
| --- | ---: | --- |
| MVP Set | 20 题 | 跑通全链路、修 schema |
| Pilot Set | 100 题 | 修标注规范、校准错因原子 |
| Function/Derivative Set | 100 题 | 验证首错定位和 VerifierTrace |
| Geometry Set | 60-100 题 | 评估 VisualEvidence 与 Geometry Lab |
| Main Set | 300-500 题 | 正式主实验 |
| Student Trial | 30-100 人 | 探索 Self-Repair、Variant Transfer、Recurrence Reduction |

## 4. 必备字段

每道题至少记录：

- `problem_id`
- `source`
- `topic`
- `difficulty`
- `corrected_text`
- `formula_evidence`
- `visual_evidence`
- `conditions`
- `student_steps`
- `gold_first_wrong_step`
- `gold_atoms`
- `gold_evidence_ids`
- `verification_targets`
- `verifier_traces`
- `need_visual`
- `socratic_policy_mode`
- `self_repair_success`
- `variant_transfer_records`
- `teacher_note`

## 5. Baseline

| Baseline | 定义 |
| --- | --- |
| Direct LLM | 不检索、不验证 |
| Study-style prompting only | 只用学习型提示词，不接诊断工作流 |
| OCR + Text RAG | 纯文本切块检索 |
| OCR + Formula RAG | 文本 + 公式联合检索 |
| Structured RAG without verifier | 有 MPES，无 verifier |
| Math-SEARAG without LearnerMemory | 有诊断，无长期画像 |
| Math-SEARAG without SocraticPolicy | 有诊断，无策略化引导 |
| Full Math-SEARAG Learning Agent | MPES + VerifierTrace + SocraticPolicy + LearnerMemory + RemediationLoop |

## 6. 指标

| 指标 | 含义 |
| --- | --- |
| First Wrong Step Accuracy | 首错定位准确率 |
| Misconception Atom Accuracy / F1 | 错因原子识别准确率 |
| Evidence Faithfulness | 解释是否被 evidence/verifier 支持 |
| Verifier Pass Rate | 关键 claim 通过验证比例 |
| Unsupported Step Rate | 无证据支持步骤比例 |
| Self-Repair Rate | 学生经追问后自我修正比例 |
| Variant Transfer Rate | 同因变式迁移成功率 |
| Recurrence Rate Reduction | 复发率下降 |
| Teacher Agreement / Cohen's Kappa | 教师标注一致性 |
| Geometry Relation Accuracy | 几何关系识别/确认准确率 |
