# 开源项目升级清单：Math-SEARAG 高中数学思维导师

本清单对应工程文档 `agent-engineering/frontend/docs/open-source-agent-upgrade-shortlist-20260613.md`。

## 最值得马上吸收的项目

1. Pix2Text：用于草稿纸中的题干、步骤、公式混合 OCR。
2. PaddleOCR：用于中文题干、行文本、手写批注识别。
3. LaTeX-OCR / pix2tex：用于公式 crop 转 LaTeX。
4. OpenR：用于 process supervision 和 step-aware verifier。
5. langchain-ai/agent-chat-ui：用于工具调用、状态、interrupt/review/resume、artifact 的 UI 思路。

## 已经改造成项目代码的能力

- OpenR 思路：新增 `process-supervision-calibration.ts`，让首错定位从“单个门禁失败”升级为“多候选首错 + margin 置信校准 + 人工标注回流”。
- 开源能力治理：新增 `open-source-capability-registry.ts`，记录项目来源、许可证、适配方式和风险。
- agent-chat-ui 思路：新增 assistant-ui 风格的 `AgentProcessRibbon`，展示 Step Alignment、Tool Calls、VerifierTrace、Review/Resume。

## 原则

- assistant-ui 仍然是前端主框架。
- TypeScript 仍然是 agent/workflow/UI 主控语言。
- Python 只保留 OCR、SymPy、几何计算、Lean/形式化验证等 TS 不擅长的能力。
- 不直接复制未审计许可证的源码；先改写接口和机制。
